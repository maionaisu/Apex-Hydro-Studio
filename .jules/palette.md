## 2026-05-11 - UI Accessibility: Buddy Links & Emoji Screen Reader Polish\n**Learning:** In PyQt6, `QFormLayout.addRow(QLabel, widget)` does NOT auto-set the buddy, preventing screen readers from linking labels to inputs. Additionally, screen readers struggle with emojis in button text, but stripping the entire astral plane can accidentally remove CJK/historical scripts.\n**Action:** Manually call `QLabel.setBuddy(widget)` when using custom labels in form layouts, and use precise regex emoji blocks (e.g., `[\\U0001F300-\\U0001FAFF...]`) in custom components like `ModernButton` to safely strip emojis for `setAccessibleName()`.
