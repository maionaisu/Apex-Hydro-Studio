## 2024-04-22 - Lazy iteration for finding metadata header\n**Learning:** In ui/views/modul3_tide.py, `f.readlines()` was used to find a skip line, which allocates an array for all lines of the file in memory. Large tide data files can use significant memory if all lines are read into memory at once.\n**Action:** Use a lazy iteration loop `(next(i for i, l in enumerate(f) if ...))` over the file descriptor directly instead.
