## 2024-05-24 - Improve Form Layout & Button Accessibility
**Learning:** In composite layout widgets like `FormRow` (which pair a `QLabel` with an input widget), the label isn't inherently linked to the input for screen readers. In PyQt6, this is fixed by calling `lbl.setBuddy(input_widget)`. Also, screen readers often mispronounce or skip reading emojis. To ensure accessible pronunciation for components like `ModernButton` with emojis, strip them with a targeted regex and set the clean string as the `AccessibleName` using `setAccessibleName(clean_name)`.
**Action:** Always link custom UI label/input pairs using `setBuddy`. Always provide clean text without emojis for screen readers using `setAccessibleName()`, ensuring standard icon-buttons maintain meaningful announcements.
